----------hướng dẫn chạy project trên IDE
B1: Chạy file ServerForm.java -> nhấp vào start để khởi động server
B2: Chạy LoginForm.java -> nhập tên để đăng kí lên server và đợi user khác.
B3: bấm vào user muốn chát hiện lên màn hình và bấm connect

---------Hướng dẫn chạy các file .jar
B1: Chạy Server.jar nhấp vào start để khởi động server
B2: Chạy Client.jar -> nhập tên để đăng kí lên server và đợi user khác.
B3: bấm vào user muốn chát hiện lên màn hình và bấm connect
